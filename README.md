# CardHub

CardHub è una piattaforma web per acquistare, vendere e gestire carte collezionabili.

## Stack didattico usato

- HTML5 per la struttura delle pagine
- CSS esterno per separare contenuto e presentazione
- Bootstrap per componenti e layout responsive
- JavaScript per validazioni e interazioni lato client
- PHP per pagine dinamiche lato server
- PostgreSQL per la persistenza dati
- JSON e JSON Schema per dati mock, API e validazione strutturale

## Avvio rapido

Da dentro la cartella del progetto:

```bash
php -S localhost:8000
```

Aprire poi:

```text
http://localhost:8000/index.php
```

## Stato

Versione iniziale: struttura base, pagine principali, mock data, validazioni client e schema SQL iniziale.
